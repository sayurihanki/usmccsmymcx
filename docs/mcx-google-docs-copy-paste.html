MCX Google Docs Blocks

Copy These Tables Directly Into Google Docs

Each example below is already formatted as an EDS/DA.live block table with the correct merged header row and column count. Open this file in a browser, copy one table at a time, and paste into Google Docs.

  1. Copy only the table grid for the block you want.
  2. Paste it into Google Docs exactly where you want the block to live.
  3. For shell content, create separate fragment docs using the header and footer tables below.
